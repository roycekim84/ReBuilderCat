# 냥냥 유적 복원단 레벨 기획서

## 1. 문서 목적

이 문서는 **냥냥 유적 복원단(ReBuilderCat)**의 초기 출시 전 MVP에 들어갈 레벨, 유적, 복원 부위, 난이도, 해금 순서, 역사 카드 내용을 정리한다.

목표는 Codex가 seed data를 만들 때 임의로 이름과 수치를 만들지 않도록 기준을 제공하는 것이다.

---

## 2. MVP 레벨 구성 요약

MVP 초기 유적은 총 6개다.

```text
Part 0. 작은 시작섬
1. 작은 항구 사당

Part 1. 아시아의 첫 유적
2. 붉은 벽돌 왕국
3. 안개의 계단 사원
4. 숲속 고대 사원
5. 정글의 붉은 탑

Part 2. 인도와 남아시아
6. 바위 왕국 유적
```

진행은 선형 구조로 시작한다.

```text
작은 항구 사당
→ 붉은 벽돌 왕국
→ 안개의 계단 사원
→ 숲속 고대 사원
→ 정글의 붉은 탑
→ 바위 왕국 유적
```

---

## 3. 공통 레벨 규칙

### 3.1 해금 규칙

다음 유적은 이전 유적을 1성 이상 클리어하면 해금된다.

```text
이전 유적 복원도 60% 이상
```

### 3.2 별 등급

| 별 | 조건 |
|---|---|
| 1성 | 복원도 60% 이상 |
| 2성 | 복원도 80% 이상 |
| 3성 | 복원도 100% |

### 3.3 부위 해금 규칙

유적 내부 부위는 순서대로 열린다.

- 첫 번째 부위는 기본 해금
- 다음 부위는 이전 부위를 최소 1회 완료하면 해금
- MVP에서는 자유 선택보다 순차 진행을 우선한다.

## 4. 난이도 티어

MVP 난이도는 4단계로 구분한다.

| 티어 | 이름 | 설명 |
|---|---|---|
| 0 | tutorial | 튜토리얼, 거의 실패 없음 |
| 1 | easy | 초반 유적 |
| 2 | normal | 기본 유적 |
| 3 | normal_plus | MVP 마지막 유적 |

Hard는 출시 후 파트에서 사용한다.

| 안개의 계단 사원 | 900~1,500 | 2,500~3,500 |

## 13. 재료 지급 기준

재료가 너무 부족하면 게임이 막힌다. 초반에는 재료를 넉넉히 지급한다.

추천:

- 튜토리얼 완료 시 재료 50 지급
- 각 유적 1성 클리어 시 다음 유적 초반 복원 가능할 정도 지급
- 광고 보급품은 5~10회 복원 가능한 양 지급

Codex는 이 문서를 기준으로 seed 파일을 작성한다.

추천 파일:

```text
lib/data/seed/seed_regions.dart
lib/data/seed/seed_relics.dart
lib/data/seed/seed_relic_parts.dart
lib/data/seed/seed_history_cards.dart
```

각 id는 이 문서의 id를 그대로 사용한다.

## 16. 레벨 기획 최종 방향

MVP 레벨은 많지 않아도 된다. 대신 각 유적의 느낌이 달라야 한다.

초기 6개 유적은 다음 감정 변화를 목표로 한다.

1. 작은 항구 사당: 시작과 튜토리얼
2. 붉은 벽돌 왕국: 첫 여행지
3. 안개의 계단 사원: 계단식 성장감
4. 숲속 고대 사원: 대표 대형 유적
5. 정글의 붉은 탑: 아시아 파트 마무리
6. 바위 왕국 유적: 다음 파트 확장 예고

이 순서를 기준으로 구현한다.
